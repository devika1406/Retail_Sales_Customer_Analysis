## Dataset Information

The dataset used in this project is the **Online Retail Transactions Dataset**  
(~541,000 records).

Due to GitHub file size limitations, the raw CSV file is not included in this repository.

You can download the dataset from Kaggle:
https://www.kaggle.com/datasets/abhishek713/online-retail

The analysis, SQL scripts, and Power BI report were built using the full dataset.
